### 제어문 (while 문 등) : 조건 지정, 진행 순서 변경
```
main(){
  int number;
  int sum;
  number = 1;
  sum = 0;
  while(number<=10)
  {
  sum = sum + number;
  number++;
}
  printf("\n 1+2+...+10=%d\n", sum);
}
```
- while문 실행 순서
1. 조건 (number<=10)을 먼저 판단
2. true인 경우 {} 수행
3. false인 경우 {} 빠져나옴
- 특징 : 조건이 false이면 한 번도 수행하지 않는다.

```
main(){
  int number;
  int sum;
  number=1;
  sum=0;
  do{
  sum=sum+number;
  number++;
}while(number<=10);
printf("\n 1+2+...+10=%d\n", sum);
}
```
- do while문 실행 순서
1. {} 먼저 수행
2. 조건 (number<=10)판단
3. true인 경우 {}수행
4. false인 경우 {}빠져나옴
- 특징 : 조건이 거짓이라도 한 번은 수행한다.
